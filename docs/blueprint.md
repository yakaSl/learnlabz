# **App Name**: LearnLabz Landing Page

## Core Features:

- Header Component: Displays the logo, navigation links (Home, Features, Pricing, About, Contact), and action buttons (Login, Get Started).
- Hero Section Component: Displays a headline, subheadline, primary CTA (Start Teaching Free), secondary CTA (I'm a Student), and a hero image of a teacher with students.
- Features Section: Showcases key features: 30-Second Enrollment (students join via link/QR), Pay 5%, Not Fixed Fees (only pay when you earn), Teach Anywhere (personal classes + multiple institutes).
- Testimonials Section: Displays testimonial cards from tutors.
- Footer Component: Includes links (About, Contact, Terms, Privacy), social media icons, and copyright information.

## Style Guidelines:

- Primary color: Teal (#008080) to represent knowledge, trustworthiness, and growth.
- Background color: Very light teal (#E0F8F8), a desaturated shade of the primary color.
- Accent color: Yellow-green (#9ACD32), an analogous color to teal, will make elements like buttons stand out.
- Body and headline font: 'PT Sans' for a humanist, modern feel.
- Use clean and professional icons to complement the educational theme. Book and student icons are possible options.
- Use a clean and structured layout to ensure easy navigation and readability. Employ a grid system to align elements consistently.
- Subtle animations when loading content and smooth transitions between sections for a professional feel.
